# ☕ Java CoffeeHouse – Front-End Web Project

A modern, responsive Coffee House website built using HTML, CSS, and JavaScript.  
This project demonstrates front-end development skills, UI design principles, and responsive web development practices.

## 🌐 Live Demo

Explore the live version of the project here:  
[View Java CoffeeHouse](https://harpreetkaur2201.github.io/java-coffeehouse/)


## 📌 Project Overview

Java CoffeeHouse is a static website designed to simulate a real café website. It focuses on delivering a clean user experience with an organized layout, visually appealing design, and responsive structure across devices.

This project was developed as part of my learning journey in front-end web development.

## ✨ Key Features

- Fully responsive layout (mobile, tablet, desktop)
- Clean and modern UI design
- Structured menu section for coffee and food items
- Interactive elements using JavaScript
- Smooth navigation and user-friendly interface

## 🛠️ Technologies Used

- HTML5 – Page structure and content  
- CSS3 – Styling and responsive design  
- JavaScript – Basic interactivity and dynamic behavior
