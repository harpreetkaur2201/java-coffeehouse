## 📌 Project Overview

Java CoffeeHouse is a static website designed to simulate a real café business website. The focus of this project was to create a visually appealing and responsive interface that works smoothly across all devices.

This project was built as part of my front-end development learning journey, helping me strengthen my understanding of layout design, responsiveness, and interactive web elements.

## ✨ Key Features

- Fully responsive layout (mobile, tablet, desktop)
- Clean and modern UI design
- Structured menu section for coffee and food items
- Interactive elements using JavaScript
- Smooth navigation and user-friendly interface

## 🛠️ Technologies Used

- HTML5 – Page structure and content  
- CSS3 – Styling and responsive design  
- JavaScript – Basic interactivity and dynamic behavior
